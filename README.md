# mcomm
Mithril Communication Matrix

# Dev Env

## setup
### install dependencies
````
npm install mithril --save-prod
npm install @types/mithril --save-dev
````

### hot reload with budo
````
npm install budo -g
````


### start with budo
````
npm start
````
meaning
````
"start": "budo --live --open index.js"
````
